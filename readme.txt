== Description ==

Get full control over your registration process by enabling Invitation based Registration on your site.

Why is this a great Idea?

# You will know know more about your user
Get a transparent picture about the members of your site and where they are combing from.


# Truly secure and transparent
Block unwanted registrations and spam once and for all with a invite only form element.

#Use Cases
Make your site launch more attractive
Keep your member flow transparent
Create a homogeneously user base

#How it works.
You create the first bunch of invite codes and define how many new codes get created for a new user.
Than the new user get a list of invite codes and can by him self invite finds.

If the friend register with the invite code he will again get new invite codes generated and so on.

Wit this technique you can quickly build a user base of one particular user type.


== Installation ==
Install like any other WordPress plugin.
Upload the entire folder to the /wp-content/plugins/ directory or install the plugin through the WordPress plugins screen directly.
Activate the plugin through the 'Plugins' menu in WordPress.
Have fun!
